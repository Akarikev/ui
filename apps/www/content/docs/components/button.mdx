## Usage

<LibraryCodeBlock baseCode="import { Button } from '@/components/ui/button'" radixCode="import { Button } from '@/components/ui/button'" language="tsx" />

Displays a button or a component that looks like a button.

Primary actions. Use variant=outline for secondary actions, variant=ghost for tertiary.

## Composition

```text
Button
├── DialogFooter
├── CardFooter
```
