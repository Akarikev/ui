Displays a list of options for the user to pick from.

Wrap options in SelectGroup. Always include SelectValue in SelectTrigger.

## Composition

```text
Select
├── Select
├── SelectTrigger
├── SelectValue
├── SelectContent
├── SelectGroup
├── SelectLabel
├── SelectItem
```
